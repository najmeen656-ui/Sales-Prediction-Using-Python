# Sales Prediction Using Python

This repository contains a Machine Learning model built to forecast sales based on advertising expenditures across different platforms.

## 📊 Project Overview
Sales prediction involves forecasting the amount of a product that customers will purchase by analyzing factors like advertising costs, target audience, and platform selection. This model helps businesses optimize their advertising strategies and maximize sales potential.

## 🚀 Model Performance
- **Algorithm Used:** Linear Regression / Machine Learning Regression
- **R-squared Score (Accuracy):** 90.59%

## 🛠️ Tech Stack & Libraries
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib / Seaborn (for data visualization)
